This is a website that I have created for you to look at whenever you are feeling down/angry/stressed and it will hopefully cheer you up

Wow that is so cool and the best gift anyone has ever given me! But how do I use it?

I know right. It really is. It's super easy just look below where you have the six different categories and pick one that you would like to have a look at. It will fetch 12 random images in that category and display them to you. The images should be (mostly) different each. The images that it has to randomly select from is updated very regularly so you should get different images every time you visit.
